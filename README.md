# Bootstrap React App
This is a bootstrap app that comes out of the box with support for:
- React
- CSS Modules
- Standard
  - Runs on a pre-commit hook
  - `linter.js` runs tests only on staged files

### Running
Install dependencies - either `npm i` or `yarn`

Then, to run locally:
```
npm run start
```
### Tests
```
npm run test
```
